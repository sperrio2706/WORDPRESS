== Description ==


DW-Medieval is a fully responsive design. It is a medieval/gothic style design for gaming and more.

DW-Medieval features optional sidebars, center block, and a javascript clock. The PRO version has many more features. Visit our website, http://www.designwicked.com, for more details!
